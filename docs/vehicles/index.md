# Vehicles

PSCM firmware documentation organized by vehicle. Not all of these are cross-compatible — the **2022 Escape** is the only vehicle whose cal data we've confirmed as layout-compatible with the 2025 Transit.

| Vehicle | PSCM prefix | Compatible with Transit 2025? |
|---|---|---|
| [2025 Transit](transit-2025.html) | `KK21` / `LK41` | — (primary target) |
| [2026 Transit](transit-2026.html) | `RK31` | ❌ different platform |
| [2022 Escape](escape-2022.html) | `LX6C` | ✅ cal layout matches — our LCA donor |
| [2024 Escape](escape-2024.html) | `PZ11` | ⚠ unconfirmed |
| [2022 F-150](f150-2022.html) | `ML34` / `ML3V` | ❌ different vendor, different cal layout |
| [2021 F-150 Lariat 502A BlueCruise](f150-2021-lariat.html) | `ML34` / `ML3V` (newer revs) | ❌ same platform as 2022 F-150, not Transit-compatible |
